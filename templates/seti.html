SETI

SETI is the Sanskrit E-Text Inventory, or "Search for E-Textual Intelligence." It harkens back to the original "register" function of GRETIL, to simply indicate in an organized fashion where exactly online searchable Sanskrit e-texts exist.

The basis of the project is a simple spreadsheet: the SETI master document. For more details, see this blog post.

Pāṇḍitya is the primary intended use case for SETI's e-text link information. Pāṇḍitya serves this info through its API (see Swagger for `/seti` routes) and provides easy-to-use hyperlinks in its interactive graphs. Look for Work nodes highlighted in gold, and explore available e-text options under the "View on" node submenu.
